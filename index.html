Game Engine Testbed by merlinblack

Simple game engine for testing ideas and techniques.

One man hobby project. Aiming at FPS or RPG type games, but is very generic at present. Still in early development as of Feb 2010. Might be used for a point and click adventure game as well.

Developed on Linux, using Vim, bash, Code::Blocks, recently Qt Creator, and of course GNU g++.

Should run on Linux, Windows, and Mac OS X however only Linux testing as been done so far.

Dependencies

Ogre3d 1.7+

Lua 5.1

Luabind 0.9

Install

Uses CMake. See README for more info.
